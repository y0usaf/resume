# Resume Repository

Personal resume repository for Sam Yousaf.

## Contents

- `resume.tex` - LaTeX source file
- `resume.pdf` - Compiled PDF version
- `CLAUDE.md` - Resume optimization guide and strategies

## Usage

### Generate PDF

```bash
pdflatex resume.tex
```

### Clean LaTeX Files

```bash
rm -f *.aux *.log *.out
```

## Repository Management

This repository uses Jujutsu (jj) for version control.

### Common Commands

```bash
# Check status
jj status

# Add changes
jj commit -m "Update resume"

# Push to remote
jj git push
```
